Примеры команд Django

Команда для создания нового проекта Django 
django-admin startproject myproject

Эта команда создаст новую директорию с именем myproject, содержащую все необходимые файлы для вашего Django проекта.

Команда для создания нового приложения в Django 
python manage.py startapp myapp

Эта команда создаст новое приложение с именем myapp в вашем проекте. Приложения используются для организации кода по функциональным модулям.

Команда для запуска сервера разработки Django 
python manage.py runserver

Эта команда запускает встроенный сервер разработки Django, который позволяет вам тестировать и просматривать ваш проект локально по адресу http://127.0.0.1:8000/.
